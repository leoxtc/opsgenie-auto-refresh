Privacy Policy for Opsgenie Alert Auto Refresher

Last updated: July 11, 2025

Summary: This extension does not collect, store, or transmit any personal data or user information. It only performs automatic page refreshes on Opsgenie alert pages.

1. Information We Collect

The Opsgenie Alert Auto Refresher extension does not collect any personal information or user data. Specifically:

  • No personal identification information
  • No browsing history or website data
  • No login credentials or authentication tokens
  • No usage analytics or tracking data
  • No cookies or local storage data

2. How the Extension Works

The extension operates entirely within your browser and:

  • Only runs on Opsgenie alert list pages (*.opsgenie.com/alert/list)
  • Automatically clicks the active search filter every 30 seconds to refresh the alert list
  • Checks for certain page elements to determine if refreshing is appropriate
  • Does not access, read, or modify any user data on the pages

3. Data Storage and Transmission

This extension:

  • Does not store any data locally on your device
  • Does not transmit any data to external servers
  • Does not communicate with any third-party services
  • Does not use any analytics or tracking services

4. Permissions Used

The extension uses the following permissions for its functionality:

  • activeTab: Allows the extension to interact with the current Opsgenie tab
  • Content Scripts: Enables the auto-refresh functionality on specified Opsgenie pages

These permissions are used solely for the extension's core functionality and not for data collection.

5. Third-Party Services

This extension does not integrate with, share data with, or communicate with any third-party services, including:

  • Analytics services
  • Advertising networks
  • Social media platforms
  • External APIs or databases

6. Changes to This Privacy Policy

We may update this privacy policy from time to time. Any changes will be reflected by updating the "Last updated" date at the top of this policy. We encourage you to review this policy periodically.

7. Contact Information

If you have any questions about this privacy policy or the extension, please contact us through:

  • GitHub Issues: https://github.com/leoxtc/opsgenie-auto-refresh

8. Legal Compliance

This extension is designed to comply with:

  • Chrome Web Store Developer Program Policies
  • Google's Privacy Requirements for Chrome Extensions
  • General Data Protection Regulation (GDPR) where applicable
Important Note: Since this extension does not collect any user data, most privacy regulations do not apply. However, we maintain this policy for transparency and compliance with Chrome Web Store requirements.
